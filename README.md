# Proyecto de análisis de ventas de SQL

## Objetivo del proyecto
Analice datos de ventas utilizando SQL Server para generar información comercial clave y KPI para la toma de decisiones.

## Herramientas utilizadas
- Microsoft SQL Server
- SQL Server Management Studio (SSMS)

## Estructura de la base de datos
- clientes
- productos
- ventas

## Análisis clave realizados
- Cálculo de ventas totales
- Ventas por cliente
- Ventas por categoría de producto
- Productos más vendidos
- Valor promedio del ticket
- ​​Tendencias de ventas mensuales

## Habilidades SQL demostradas
- Modelado de bases de datos
- Operaciones JOIN
- Agregaciones (SUM, AVG)
- GROUP BY, ORDER BY
- Generación de KPI

Este proyecto demuestra habilidades prácticas de SQL aplicadas a escenarios de análisis de ventas del mundo real.
